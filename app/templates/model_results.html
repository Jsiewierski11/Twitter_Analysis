For this application I tried several different models to see what performed best in sentiment classification. The first model I made for topic classification, Multinomial Naive Bayes, had an accuracy score of 96% so I decided to stick with that model for this application.
Sentiment classification was a little tricker and I experimented with a couple different models. Below you can see the accuracy and the F1 scores of the different models that I trained for this project.

Accuracy F1 Score
Naive Bayes trained on TF-IDF for Sentiment Classification 68% 68%
Naive Bayes trained on TF-IDF for Topic Classification 96% 96%
Logistic Regression trained on Doc2Vec for Sentiment Classification 39% 39%
Naive Bayes trained on Doc2Vec for Sentiment Classification 35% 22%
